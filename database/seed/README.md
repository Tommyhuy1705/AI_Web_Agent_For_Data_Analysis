# Database Seed Data

This directory contains scripts to populate the database with mock/test data.

## Usage

```bash
# Load mock sales data into raw_staging.raw_sales
python database/seed/mock_data_loader.py
```

## Files

| File | Description |
|---|---|
| `mock_data_loader.py` | Loads 500 mock sales records into `raw_staging.raw_sales` |
